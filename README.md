# AI24x7 Elderly Care AI 👴

AI-powered fall detection + 24/7 elderly monitoring. No wearable needed.

## Features

✅ **Fall Detection** (< 5 seconds)
- Camera watches elderly person 24/7
- Detects sudden falls instantly
- Sends WhatsApp alert + video clip to family

✅ **Family Alert Cascade**
1. Fall detected (< 5 sec) → WhatsApp to all family members
2. If no response in 10 min → Auto-call ambulance
3. Share location + video clip

✅ **Voice Check-in**
- Every 2 hours: "Are you okay?" in Hindi/Tamil/Telugu
- No response in 5 min → Alert sent
- Gentle, non-intrusive

✅ **Activity Monitoring**
- Daily routine tracking
- Anomaly detection (sleep pattern changes)
- Weekly report to family

✅ **Privacy-First**
- No live streaming (only alerts)
- 30-second clip sent, then deleted
- Local processing option
